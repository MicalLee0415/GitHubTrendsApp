## 🔧 自定义配置

### 配置 GitHub Token（推荐）
```typescript
// entry/src/main/ets/services/GitHubService.ets
private readonly token = 'YOUR_GITHUB_TOKEN'

// 获取 Token: https://github.com/settings/tokens
// 权限: public_repo
// 提高请求限制从 60/小时 到 5000/小时
```

### 修改应用信息
```json5
// entry/src/main/module.json5
{
  "module": {
    "name": "entry",
    "description": "你的应用描述"
  }
}
```

### 自定义颜色主题
```typescript
// entry/src/main/ets/components/ProjectCard.ets
// 修改颜色值自定义主题
fontColor('#1D1D1F')  // 主文字颜色
fontColor('#007AFF')  // 强调色
backgroundColor('#FFFFFF')  // 背景色
```
