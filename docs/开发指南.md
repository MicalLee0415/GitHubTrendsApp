```bash
Build -> Build Hap(s)/APP(s) -> Build Hap(s)

# 输出位置：
entry/build/default/outputs/default/entry-default-unsigned.hap
```

### 2. 编译 APP（用于发布）
```bash
# 在 DevEco Studio 中：
Build -> Build Hap(s)/APP(s) -> Build APP(s)

# 输出位置：
entry/build/default/outputs/default/entry-default-signed.app
```

### 3. 命令行编译
```bash
# 进入项目目录
cd GitHubTrendsApp

# 编译 HAP
./hvigorw clean
./hvigorw assembleHap --mode module -p module=entry@default -p product=default

# 编译 APP
./hvigorw assembleApp --mode project -p product=default
```

## 四、测试

### 1. 单元测试
创建测试文件：`entry/src/ohosTest/ets/test/GitHubServiceTest.ets`

```typescript
import { describe, it, expect } from '@ohos/hypium'
import { GitHubService } from '../../../main/ets/services/GitHubService'

export default function GitHubServiceTest() {
  describe('GitHubService', () => {
    const service = new GitHubService()

    it('should get daily top projects', async () => {
      const projects = await service.getDailyTopProjects()
      expect(projects.length).assertEqual(10)
      expect(projects[0].name).assertNotNull()
    })

    it('should analyze tech trend', async () => {
      const projects = await service.getDailyTopProjects()
      const trend = await service.analyzeTechTrend(projects, 'daily')
      expect(trend.summary).assertNotNull()
      expect(trend.hotTechs.length).assertLargerThan(0)
    })
  })
}
```

运行测试：
```bash
# 在 DevEco Studio 中右键测试文件 -> Run
# 或命令行
./hvigorw test --mode module -p module=entry@default
```

### 2. UI 测试
创建测试文件：`entry/src/ohosTest/ets/test/IndexTest.ets`

```typescript
import { Driver, Component } from '@ohos.UiTest'
import { describe, it, beforeAll } from '@ohos/hypium'

export default function IndexTest() {
  describe('Index', () => {
    let driver: Driver

    beforeAll(() => {
      driver = Driver.create()
    })

    it('should display project cards', async () => {
      // 等待数据加载
      await driver.delayMs(2000)
      
      // 查找项目卡片
      const card = await driver.findComponent(ON.text('查看详情'))
      expect(card).assertNotNull()
    })

    it('should switch tabs', async () => {
      // 点击"本周趋势"标签
      const weeklyTab = await driver.findComponent(ON.text('本周趋势'))
      await weeklyTab.click()
      
      await driver.delayMs(1000)
      
      // 验证是否切换成功
      const activeTab = await driver.findComponent(ON.text('本周趋势'))
      expect(activeTab).assertNotNull()
    })
  })
}
```

## 六、发布流程

### 1. 准备发布材料

#### 应用图标
```
entry/src/main/resources/
├── base/media/
│   ├── icon.png (512x512)
│   └── startIcon.png (256x256)
└── rawfile/
    └── privacy_policy.html (隐私政策)
```

#### 应用截图
```
准备至少 3 张截图：
- 首页展示
- 项目列表
- 趋势分析
尺寸：1080x1920 或更高
```

#### 应用描述
```
应用名称：GitHub 开源趋势
应用简介：追踪 GitHub 热门项目和 AI 编程工具趋势

功能特点：
• 每日获取 GitHub 星数 Top 10 项目
• 每周技术趋势分析和学习建议
• AI 编程工具动态追踪
• 苹果风格的精美界面

关键词：GitHub, 开源, 趋势, AI, 编程工具
```

### 3. 上传到华为应用市场

#### 步骤：
```bash
1. 登录 AppGallery Connect
2. 选择"我的应用"
3. 点击"创建应用"
4. 填写应用信息：
   - 应用名称：GitHub 开源趋势
   - 应用包名：com.github.trends
   - 应用类型：HarmonyOS 应用
   
5. 上传安装包
   - 上传 .app 文件
   - 等待校验完成
   
6. 填写应用详情
   - 上传图标和截图
   - 填写应用描述
   - 设置应用分类
   
7. 配置隐私政策
   - 上传隐私政策文档
   - 填写权限说明
   
8. 提交审核
   - 检查所有信息
   - 点击"提交审核"
   - 等待审核（通常 3-5 个工作日）
```

### 4. 审核要点
- ✓ 应用无崩溃和严重 Bug
- ✓ UI 美观，交互流畅
- ✓ 功能符合描述
- ✓ 隐私政策完整
- ✓ 网络权限说明清晰
- ✓ 无违法违规内容

## 八、常见问题

### Q1: 签名失败
```
解决方案：
1. 检查华为账号是否登录
2. 检查网络连接
3. 清除 DevEco Studio 缓存：File -> Invalidate Caches
4. 重新生成签名
```

### Q2: 安装失败
```bash
# 检查设备连接
hdc list targets

# 检查应用签名
hdc shell bm dump -n com.github.trends

# 查看错误日志
hdc shell hilog | grep -i install
```

### Q3: 网络请求失败
```
1. 检查是否添加了 INTERNET 权限
2. 检查网络连接
3. 检查 HTTPS 证书
4. 查看网络日志
```

### Q4: 应用闪退
```bash
# 查看崩溃日志
hdc shell hilog | grep -i crash

# 查看应用日志
hdc shell hilog -t GitHubTrendsApp
```
