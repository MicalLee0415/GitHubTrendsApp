# HarmonyOS 开发环境搭建指南

## 一、下载安装 DevEco Studio

### 1. 下载 DevEco Studio
- 访问官网：https://developer.harmonyos.com/cn/develop/deveco-studio/
- 选择 Linux 版本（适用于 UOS 系统）
- 下载 `deveco-studio-4.0.0.600-linux.tar.gz`（或最新版本）

### 2. 安装步骤

```bash
# 解压安装包
cd ~/Downloads
tar -xzf deveco-studio-*.tar.gz

# 移动到安装目录
sudo mv deveco-studio /opt/

# 添加执行权限
sudo chmod +x /opt/deveco-studio/bin/studio.sh

# 创建桌面快捷方式
cat > ~/.local/share/applications/DevEco-Studio.desktop <<EOF
[Desktop Entry]
Name=DevEco Studio
Comment=HarmonyOS IDE
Exec=/opt/deveco-studio/bin/studio.sh
Icon=/opt/deveco-studio/bin/studio.png
Terminal=false
Type=Application
Categories=Development;IDE;
EOF

# 启动 DevEco Studio
/opt/deveco-studio/bin/studio.sh
```

### 3. 首次启动配置
1. 选择 UI 主题（推荐 Dark）
2. 配置 Node.js 路径（DevEco 会自动下载）
3. 配置 HarmonyOS SDK 路径
4. 接受许可协议

## 二、注册华为开发者账号

### 1. 注册流程
1. 访问：https://developer.huawei.com/consumer/cn/
2. 点击"注册"
3. 使用手机号注册
4. 完成邮箱验证

### 2. 实名认证
1. 登录开发者中心
2. 进入"个人中心" -> "实名认证"
3. 上传身份证照片
4. 等待审核（通常 1-2 个工作日）

### 3. 创建应用
1. 进入"AppGallery Connect"
2. 点击"我的项目" -> "创建项目"
3. 填写项目信息：
   - 项目名称：GitHubTrendsApp
   - 项目类型：应用
4. 添加应用：
   - 应用名称：GitHub 开源趋势
   - 应用类型：HarmonyOS 应用
   - 应用包名：com.github.trends
5. 记录 App ID 和 App Secret

## 三、配置签名证书

### 方式一：自动签名（推荐）
1. 在 DevEco Studio 中打开项目
2. 点击 "File" -> "Project Structure"
3. 选择 "Project" -> "Signing Configs"
4. 勾选 "Automatically generate signature"
5. 登录华为账号
6. 点击 "Apply" 自动生成签名

### 方式二：手动签名
```bash
# 1. 生成密钥对
keytool -genkeypair -alias githubtrends -keyalg RSA -keysize 2048 -validity 36500 -keystore githubtrends.p12 -storetype PKCS12

# 2. 生成证书请求文件
keytool -certreq -alias githubtrends -keystore githubtrends.p12 -file githubtrends.csr

# 3. 在 AppGallery Connect 上传 CSR，下载证书
# 4. 在 DevEco Studio 中配置签名
```

## 四、配置 HarmonyOS SDK

### 1. SDK 安装
1. DevEco Studio 启动后，会自动提示下载 SDK
2. 选择 SDK 版本：API 10 (HarmonyOS 4.0+)
3. 选择安装路径
4. 等待下载完成

### 2. SDK 组件
确保安装以下组件：
- HarmonyOS SDK
- HarmonyOS NDK
- HarmonyOS Toolchains
- Previewer

## 五、项目配置

### 1. 修改包名
编辑 `entry/build-profile.json5`:
```json5
{
  "api": {
    "compatible": 10,
    "target": 10
  },
  "buildOption": {
    "strictMode": {
      "caseSensitiveCheck": true
    }
  },
  "buildOptionSet": [
    {
      "name": "release",
      "arkOptions": {
        "obfuscation": {
          "ruleOptions": {
            "enable": false
          }
        }
      }
    }
  ],
  "targets": [
    {
      "name": "default"
    }
  ]
}
```

### 2. 配置网络权限
编辑 `entry/src/main/module.json5`:
```json5
{
  "module": {
    "name": "entry",
    "type": "entry",
    "description": "$string:module_desc",
    "mainElement": "EntryAbility",
    "deviceTypes": [
      "default",
      "tablet"
    ],
    "requestPermissions": [
      {
        "name": "ohos.permission.INTERNET",
        "reason": "需要访问网络获取 GitHub 数据"
      }
    ],
    "deliveryWithInstall": true,
    "installationFree": false,
    "pages": "$profile:main_pages",
    "abilities": [
      {
        "name": "EntryAbility",
        "srcEntry": "./ets/entryability/EntryAbility.ets",
        "description": "$string:EntryAbility_desc",
        "icon": "$media:icon",
        "label": "$string:EntryAbility_label",
        "startWindowIcon": "$media:startIcon",
        "startWindowBackground": "$color:start_window_background",
        "exported": true,
        "skills": [
          {
            "entities": [
              "entity.system.home"
            ],
            "actions": [
              "action.system.home"
            ]
          }
        ]
      }
    ]
  }
}
```

## 六、连接真机设备

### 1. 开启开发者模式
在平板上：
1. 设置 -> 关于平板
2. 连续点击"HarmonyOS 版本" 7 次
3. 输入密码开启开发者模式

### 2. 启用 USB 调试
1. 设置 -> 系统和更新 -> 开发者选项
2. 开启"USB 调试"
3. 开启"仅充电模式下允许 ADB 调试"

### 3. 连接设备
```bash
# 连接平板到电脑（USB）
# 在 DevEco Studio 中会自动识别设备
```

### 4. 授权调试
- 平板上会弹出授权对话框
- 勾选"始终允许"
- 点击"允许"

## 七、验证环境

### 1. 创建测试项目
1. DevEco Studio -> File -> New -> Create Project
2. 选择 "Empty Ability"
3. 填写项目信息
4. 点击 Finish

### 2. 运行测试
1. 连接平板或启动模拟器
2. 点击运行按钮（绿色三角形）
3. 如果能看到 "Hello World"，环境搭建成功

## 八、常见问题

### Q1: DevEco Studio 启动失败
```bash
# 检查依赖
sudo apt-get install openjdk-11-jdk
sudo apt-get install libcanberra-gtk-module

# 检查环境变量
echo $JAVA_HOME
```

### Q2: 设备连接不上
```bash
# 检查 USB 连接
lsusb

# 重启 ADB 服务
adb kill-server
adb start-server
adb devices
```

### Q3: 签名失败
- 检查华为账号是否登录
- 检查网络连接
- 尝试重新生成签名

### Q4: SDK 下载失败
- 检查网络连接
- 配置代理（如需要）
- 更换下载镜像
