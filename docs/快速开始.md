# 快速开始指南

## 🎯 项目概述
这是一个运行在 HarmonyOS 4.2.0.213 上的应用，用于追踪 GitHub 热门开源项目和 AI 编程工具趋势。

## 📋 前置准备

### 必需条件
- ✅ HarmonyOS 平板设备（系统版本 4.2.0.213 或更高）
- ✅ Linux/UOS 电脑（用于开发）
- ✅ 华为开发者账号（免费注册）
- ✅ USB 数据线

### 技能要求
- 基础 TypeScript/JavaScript 知识
- 了解基本的移动应用开发概念

## 🚀 五步快速开始

### 第一步：安装开发工具（约 10 分钟）

```bash
# 1. 下载 DevEco Studio
# 访问：https://developer.harmonyos.com/cn/develop/deveco-studio/
# 下载 Linux 版本

# 2. 解压并安装
cd ~/Downloads
tar -xzf deveco-studio-*.tar.gz
sudo mv deveco-studio /opt/

# 3. 启动
/opt/deveco-studio/bin/studio.sh

# 4. 首次启动配置
# - 选择主题（推荐 Dark）
# - 自动下载 SDK（API 10）
# - 接受许可协议
```

### 第二步：注册开发者账号（约 5 分钟）

```bash
# 1. 访问华为开发者中心
# https://developer.huawei.com/consumer/cn/

# 2. 注册账号
# - 使用手机号注册
# - 完成邮箱验证

# 3. 实名认证
# - 上传身份证
# - 等待审核（1-2 个工作日）

# 4. 创建应用项目
# - AppGallery Connect -> 我的项目 -> 创建项目
# - 项目名称：GitHubTrendsApp
# - 添加应用，记录 App ID
```

### 第三步：导入项目（约 3 分钟）

```bash
# 1. 打开 DevEco Studio
# 2. File -> Open
# 3. 选择 GitHubTrendsApp 目录
# 4. 等待项目同步

# 项目结构：
GitHubTrendsApp/
├── entry/src/main/ets/
│   ├── pages/Index.ets          # 主页面
│   ├── components/              # UI 组件
│   ├── services/GitHubService.ets  # 数据服务
│   └── model/DataModel.ets      # 数据模型
└── docs/                        # 开发文档
```

### 第四步：配置签名（约 2 分钟）

```bash
# 1. 在 DevEco Studio 中：
#    File -> Project Structure -> Signing Configs

# 2. 勾选 "Automatically generate signature"

# 3. 登录华为账号

# 4. 点击 Apply
```

### 第五步：运行应用（约 2 分钟）

```bash
# 1. 在平板上开启开发者模式
#    设置 -> 关于平板 -> 连续点击"HarmonyOS 版本" 7 次

# 2. 启用 USB 调试
#    设置 -> 系统和更新 -> 开发者选项 -> USB 调试

# 3. 连接平板到电脑
#    - 授权调试
#    - 勾选"始终允许"

# 4. 在 DevEco Studio 中：
#    - 选择设备（你的平板）
#    - 点击运行按钮（绿色三角形）
#    - 应用会自动安装并启动
```

## 🛠️ 自定义配置

### 配置 GitHub Token（提高 API 限制）

```typescript
// entry/src/main/ets/services/GitHubService.ets
private readonly token = 'YOUR_GITHUB_TOKEN' // 替换为你的 Token

// 获取 Token：https://github.com/settings/tokens
// 权限：public_repo
```

### 修改应用包名

```json5
// entry/build-profile.json5
{
  "bundleName": "com.yourcompany.githubtrends" // 修改为你的包名
}
```

### 自定义主题颜色

```typescript
// entry/src/main/ets/components/ProjectCard.ets
// 修改颜色值：
fontColor('#1D1D1F')  // 文字颜色
backgroundColor('#FFFFFF')  // 背景颜色
```

## 🔧 常见问题

### Q: DevEco Studio 启动失败？
```bash
# 检查依赖
sudo apt-get install openjdk-11-jdk
sudo apt-get install libcanberra-gtk-module
```

### Q: 设备连接不上？
```bash
# 重启 ADB
hdc kill-server
hdc start-server
hdc list targets
```

### Q: 编译失败？
```bash
# 清理项目
Build -> Clean Project

# 重新同步
File -> Sync Project with Build Files
```

### Q: 网络请求失败？
```typescript
// 检查权限配置
// entry/src/main/module.json5
"requestPermissions": [
  {
    "name": "ohos.permission.INTERNET"
  }
]
```
